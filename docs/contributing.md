# Contributing

[![Build Status](https://travis-ci.org/jelmer/xandikos.png?branch=master)](https://travis-ci.org/jelmer/xandikos)
[![Windows Build Status](https://ci.appveyor.com/api/projects/status/fjqtsk8agwmwavqk/branch/master?svg=true)](https://ci.appveyor.com/project/jelmer/xandikos/branch/master)

Contributions to Xandikos are very welcome. If you run into bugs or have
feature requests, please file issues [on GitHub](https://github.com/jelmer/xandikos/issues/new).
If you're interested in contributing code or documentation, please read
[CONTRIBUTING](https://github.com/jelmer/xandikos/blob/master/CONTRIBUTING.rst).
Issues that are good for new contributors are tagged
[new-contributor](https://github.com/jelmer/xandikos/labels/new-contributor) on GitHub.
